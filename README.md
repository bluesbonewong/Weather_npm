## Installing
Using npm:
```
$ npm install -g bruce-weather
```
## Example
get the your city's weather
```
$ weather
当前城市是：合肥市
时间为：周六 03月10日 (实时：16℃)
PM2.5为：58
温度为：17 ~ 8℃ 东南风微风
天气情况：晴转多云
```
get the city's weather that you want
```
$ weather 马鞍山市
当前城市是：马鞍山市
时间为：周六 03月10日 (实时：14℃)
PM2.5为：68
温度为：17 ~ 5℃ 南风微风
天气情况：多云
```
